Twitch Point Integration で利用可能なイベントを掲載しています。

|       項目名       |                                                                    説明                                                                     |
| :----------------: | :-----------------------------------------------------------------------------------------------------------------------------------------: |
|       指定名       |                                                            Twitch に記載する名前                                                            |
|       レベル       | どのぐらい Noita に対して強力かを表しています。0-9 の 10 段階評価となります。<br /> Twitch のポイント報酬値を決める際の参考にしてください。 |
| RandomAll 重み付け |            RandomAll のポイント報酬が呼ばれたときに参照される重み付けです。<br />この値が高ければ高いほど排出率が高くなります。             |
|      効果時間      |                                                         どのぐらい効果が継続するか                                                          |
|    テキスト入力    |                                    「視聴者にテキストの入力を要求」の項目が必要かどうかを記載しています                                     |

## AntiGravity

放射物が徐々に上へと飛んでいくので、狙いづらくなります。

|      効果時間      |    10 秒    |
|    テキスト入力    |    不要     |

## MovementControl

視聴者が指定した方向に Noita を強制移動させます。  
強制移動させられるので、溶岩に突っ込ませたり位置を固定したりとさまざまな悪用方法が考えられます。

入力枠に対して「up, down, right, left」以外を入力した場合は、ランダムで抽選されます。

|       項目名       |              値               |
| :----------------: | :---------------------------: |
|       指定名       |        MovementControl        |
|       レベル       |               8               |
| RandomAll 重み付け |              60               |
|      効果時間      |             3 秒              |
|    テキスト入力    | 必要（up, down, right, left） |

## PotionGift

Noita の周りにポーションを召喚します。  
ポーションは通常のポーション排出と同じなため、アンブロシア等の有利なポーションがでることもあります。
ポーションは 2 つ、3 つ召喚される場合があります。

- 5%: ポーション 3 つ
- 5%: ポーション 2 つ
- 90%: ポーション 1 つ
